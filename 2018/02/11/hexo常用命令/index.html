常用命令

发表于 2018-02-11 | 分类于 杂记 |

以前时长会用到一些命令，但是时间一长不用就忘了，今天将这些命令整理一下，用作参考

Mac(Linux)

  • shift+command+. :显示隐藏文件命令
  • alt+command+c :复制当前的文件路径
  • ping www.baidu.com :获取当前网站的ip地址
  • hostname :显示主机名称
  • ifconfig :显示主机IP地址
  • top :显示进程、load、cpu、内存等；Ctrl+ C 退出(control+c)Mac
  • vi :编辑一个文件

  • control+c :结束当前开启的进程

  • ps aux :查看所有进程

  • ps aux | grep java :查看所有Java进程
  • grep NullPoiterException xxx.log | wc -l :日志文件中搜索NPE异常有多少个
  • grep NullPoiterException topservice.log > npe.log :日志文件中搜索NPE异常到npe.log
  • cat pom.xml :查看文件内容（全部）
  • more pom.xml :查看文件内容（分屏）
  • cat pom.xml | grep upp :查看包含spring的依赖
  • file /bin/ls :查看机器是否为64位机器

  • objdump -a xxx.so :查看so是否为64位

  • ls :显示当前目录下的所以文件

  • ls :-l 等同于ll

  • ll

  • env :查看环境变量

  • env | grep LANG : 查看特定环境变量
  • cd :返回用户根目录
  • cd – :返回上一次进入的目录
  • 向上箭头，运行上一个命令

  • 向下箭头，运行下一个命令

  • mv x.so y.so :重命名

  • wget url :下载或者访问url
  • curl url :访问url
  • pwd :查看当前所在目录
  • scp -r .bash_profile admin@10.232.13.132:~ :远程拷贝
  • ssh username@hostname :SSH登录到另一台服务器

Git

  • git branch dev :创建分支
  • git checkout :切换分支

  • git push –set-upstream :上传代码

  • git status :查看常用命令

  • git diff :查看变更内容
  • git log :查看提交历史

git更改用户和邮箱命令

查看git用户和邮箱

  • git config user.name
  • git config user.email

修改git用户和邮箱

  • git config - -global user.name
  • git config - -global user.email

GitBook

  • npm install -g gitbook :安装gitbook
  • gitbook -V :查看gitbook安装是否成功，V为大写
  • gitbook serve :启动gitbook，默认端口为4000
  • gitbook serve –lrport 32928 –port 4000 :指定监听端口和启动的服务端口

jenkins

  • jar -jar jenkins.war:启动
